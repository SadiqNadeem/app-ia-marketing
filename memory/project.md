# Proyecto: SaaS de Marketing con IA para Negocios Locales

## Descripcion
Plataforma web donde negocios locales (restaurantes, peluquerias, tiendas, gimnasios) generan contenido de marketing automaticamente usando IA. Suben fotos, la IA genera flyers, posts, historias y videos, y la app publica directamente en redes sociales.

## Stack tecnico
- Frontend: Next.js 14 con App Router y TypeScript
- Base de datos y auth: Supabase
- Hosting: Vercel
- IA texto: OpenAI API (GPT-4o)
- IA imagenes: DALL-E 3
- IA video: Runway ML / Pika (fase futura)
- Pagos: Stripe
- Estilos: Tailwind CSS + shadcn/ui
- Fuente: Inter
